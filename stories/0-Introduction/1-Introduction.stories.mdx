# Welcome to Quark Web Components

Quark Web Components is a suite of native Material Design Web Components for building line of business applications. Being native Web Components means that they can be used as-is with your favorite front-end JavaScript framework or library, such as Angular, React, Vue, Polymer, LitElement, Stencil, Svelte, etc. You can even use them in a vanilla HTML/CSS and JavaScript application, if you wish.
